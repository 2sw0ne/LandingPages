# DIGITAL CEO — Dossier de démarrage

Page de présentation à défilement (parcours d'intégration) pour Digital CEO Land.

## Héberger sur GitHub Pages
1. Crée un dépôt et dépose le contenu de ce dossier à la racine (`index.html`, `styles.css`, `script.js`, dossier `img/`).
2. Dans **Settings → Pages**, choisis la branche `main` et le dossier `/ (root)`.
3. Ton site sera en ligne à `https://<ton-pseudo>.github.io/<nom-du-depot>/`.

## Fichiers
- `index.html` — la page (textes éditables directement dans le code)
- `styles.css` — styles
- `script.js` — apparition au défilement + barre de navigation
- `img/` — toutes les images
